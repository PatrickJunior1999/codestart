# Fase 2 — Supabase Auth e Persistência Inicial

## Entregas desta fase

- Login real com Supabase Auth;
- Cadastro real com metadados do aluno;
- Consentimento LGPD no cadastro;
- Recuperação de senha;
- Modo demonstração preservado;
- Carregamento de módulos via Supabase;
- Carregamento de progresso via Supabase;
- Carregamento de lições via Supabase;
- Migration complementar para criação de profile.

## Decisão do conselho

A plataforma agora começa a sair do modo protótipo e se aproxima de uma aplicação educacional em produção. O modo demonstração continua importante para apresentação e desenvolvimento, mas a versão de aplicação real deve usar Supabase para autenticação e persistência.

## Ponto crítico

O projeto ainda não deve ser aplicado oficialmente a alunos antes da implementação de:

- correção segura dos quizzes;
- registro de atividades;
- projeto final salvo;
- regra real de certificação;
- certificado PDF com QR Code;
- testes de segurança de acesso.
