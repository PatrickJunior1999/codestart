# Fase 1 — Setup do Projeto

## Entregas desta fase

- Estrutura React + Vite + TypeScript;
- Design System inicial;
- Componentes reutilizáveis;
- Rotas públicas e protegidas;
- Layout principal;
- Páginas base;
- Dados mockados dos módulos;
- Migration SQL inicial do Supabase.

## Decisão do conselho

A base está adequada para iniciar o desenvolvimento técnico, mas ainda não deve ser considerada produção. Antes de aplicar em escola/instituição, será obrigatório conectar Supabase Auth, banco real, quizzes seguros e certificado verificável.
