# Roteiro breve de apresentação — CodeStart v1.0

## 1. Abertura

O CodeStart é um curso online autoinstrucional para introdução à lógica de programação, pensamento computacional e primeiros conceitos de programação. O sistema foi desenvolvido como um objeto digital de aprendizagem gamificado, com módulos progressivos, atividades práticas, avaliação final e certificado validável publicamente.

## 2. Problema educacional

Muitos estudantes têm o primeiro contato com programação de forma abstrata, já começando por código textual e sintaxe. Isso pode gerar desmotivação. O CodeStart propõe uma trilha gradual, começando por lógica cotidiana, evoluindo para algoritmos, pseudocódigo, blocos e código visual.

## 3. Metodologia

A trilha combina aprendizagem ativa, gamificação, feedback imediato e progressão por missões. O aluno precisa estudar, praticar e demonstrar aproveitamento antes de avançar.

## 4. Estrutura do curso

O curso possui 8 módulos, atividades práticas, quizzes, projeto final, avaliação final e certificado. A conclusão de cada módulo exige atividade prática concluída e quiz com pelo menos 70%.

## 5. Interatividade

O sistema possui atividades de ordenação, associação, pseudocódigo, programação em blocos e desenho por código visual. Na programação em blocos, o aluno monta sequências e vê o mascote LOGIC se movimentar pelo tabuleiro.

## 6. Certificação

O certificado só é liberado após conclusão dos módulos, entrega do projeto final e aprovação na avaliação final. O certificado possui código e QR Code para validação pública.

## 7. Administração

A área administrativa não transforma o sistema em uma plataforma escolar. Ela serve apenas para a equipe do projeto acompanhar evidências: alunos cadastrados, progresso, avaliações, projetos e certificados.
