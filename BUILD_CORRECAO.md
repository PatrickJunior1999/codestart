# Correção de build v1.0.1

Esta versão remove arquivos antigos do painel de professor que podiam permanecer quando a v1.0 era extraída por cima de uma pasta antiga.

Antes de rodar o build, use uma pasta limpa ou apague manualmente `src/pages/TeacherDashboardPage.tsx` se existir.

Comandos recomendados:

```bash
npm install
npm run build
```
